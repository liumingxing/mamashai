【宝宝几个月可以添加辅食】

添加辅食的时间因个体差异而定，母乳喂养的宝宝，身体素质较好，可以在六个月以后再添加辅食；而人工喂养的宝宝就需要在4-6个月的时候添加。并且要根据宝宝的身体情况和发育状况进行添加。

【过早添加辅食好吗】

如果在宝宝肠胃还不成熟的时候，就提前给宝宝添加辅食，既不能很好的消化辅食，还会对胃造成伤害；即使是给宝宝喝果汁或者蔬菜汁，也有可能造成过敏。而且这个时期宝宝还在进行母乳喂养，添加辅食还有可能影响到宝宝对母乳的需求量。

【添加辅食前宝宝会给哪些信号】

1、宝宝体重是否足够：增加辅食体重需要达到出生时的2倍，至少达到6千克。

2、宝宝是否具有想吃东西的行为：如别人在他旁边吃饭时他会感兴趣，或者宝宝将手或玩具往嘴里塞，说明他对吃饭有了兴趣。

3、宝宝的发育是否成熟：宝宝能控制头部和上半身，能够扶着或靠着坐，胸能挺起来，头能竖起来，宝宝可以通过转头、前倾、后仰等来表示想吃或不想吃，这样就不会发生强迫喂食的情况。

4、宝宝是否有吃不饱的表现：宝宝原来能一夜睡到天亮，现在却经常半夜哭闹。当宝宝在6个月前后出现生长加速期时，是开始添加辅食的最佳时机。

5、伸舌反射是否消退：很多父母都发现刚给宝宝喂辅食时，他常常把刚喂进嘴里的东西吐出来，认为是宝宝不爱吃。其实宝宝这种伸舌头的表现是一种本能的自我保护，称为“伸舌反射”，说明喂辅食还不到时候。伸舌反射一般到4个月前后才会消失。

6、宝宝尝试吃东西的行为：当爸爸妈妈舀起食物放进宝宝嘴里时，他会尝试着舔进嘴里并咽下，显得很高兴、这时你可以放心给宝宝喂食了。如果宝宝将食物吐出，把头转开或推开你的手，说明宝宝不要吃也不想吃。你一定不能勉强，隔几天再试试。

【宝宝的第一份辅食吃什么好】

添加的第一个辅食应该是纯米粉。因为米粉糊不含面筋，和其他食物相比不易引起过敏反应。

【宝宝多大开始喂蛋黄】

鸡蛋中含有一种被称为“类卵黏蛋白”的成分，很容易导致宝宝过敏。更重要的是蛋黄当中的铁元素不易吸收。一般情况下，宝宝7个月后就可以开始添加蛋黄，但要从少量开始，逐渐增加食用量。

【辅食添加的原则】

与宝宝的月龄相适应；一种到多种；辅食要鲜嫩、卫生、口味好；进食时的心情要愉快；从少量到多量；从稀到稠；从细小到粗大；婴儿添加辅食：少盐不甜；8个月之前的食物不需要加盐，8个月后婴儿添加的辅食中可少量加盐；婴儿添加辅食忌油腻；遇到不适要立刻停止添加；吃流质或泥状食品不宜过长；不可很快让辅食替代乳类。

【各类辅食的添加顺序】

1、从种类讲：应按“淀粉（谷物）—蔬菜—水果—动物”的顺序来添加。从一个种类过渡到另一个种类的时间可以是一两周。添加时要按从单一到多样的顺序进行，初次添加时不要同时给宝宝吃两三种食物。

2、从数量讲：从数量来讲，应按由少到多的顺序，一开始只是给宝宝试吃与品尝，或者说在喂奶之后试吃一点，在宝宝适应后逐渐增加。

3、从质地讲：液体（如米糊、菜水、果汁等）；泥糊（如浓米糊、菜泥、肉泥、鱼泥、蛋黄等）；固体（如软饭、烂面条、小馒头片等）

【宝宝能吃多少辅食】

刚开始时，每天喂一次辅食就可以了。最初，宝宝应该每天只吃一次半流质的米粉糊。到大约8个月左右，就可以每天吃3次辅食了，这时候典型的每日食谱可能包括：母乳或强化铁配方奶；强化铁的米粉或其他谷物粉；黄色和绿色蔬菜；水果；少量富含蛋白质的食物，如禽肉、豆腐、肉类。

【宝宝吃辅食后大便变臭了正常吗】

以前宝宝一直都是纯母乳喂养的，一旦宝宝开始吃即便很少的辅食，以前那甜甜的大便也会开始有臭味了，这是很正常的现象。如果宝宝的大便看起来有些干硬（米糊、香蕉和苹果泥能加重便秘），你可以给他喂其他水果、蔬菜、燕麦或大麦麦片试一试。

【有家族过敏史的宝宝辅食添加要注意什么】

有家族过敏史(父、母、兄、姐有气喘、过敏性鼻炎或结膜炎、异位性皮肤炎)的宝宝，可将辅食添加时间延缓到六个月，建议先从添加米粉开始。至于容易诱发过敏的食物，如甲谷类海鲜(虾、螃蟹)、坚果(花生、核桃)、柑橘、橙、芒果、奇异果、草莓、碗豆、花生及蛋白等，最好延到一岁以后再给宝宝吃了。

较不易引发过敏的食材：

1、蔬菜：淮山、土豆、胡罗卜、南瓜、青豆、马铃薯、花椰菜

2、水果：梨子、桃子、苹果

3、 五谷：米粉、小米

【不同月龄辅食添加内容与建议】

【4-6个月】品尝：宝宝绝大部分还没有长牙，泥糊状食物非常适合宝宝。添加的量从1-2勺开始，以后逐步增加。依次提供米粉（可用母乳、配方奶或苹果汁调配），蔬菜泥后，水果汁或泥（果汁先从兑水开始，然后再喝原汁），接下来是果泥。

【7-9个月】学咀嚼，练吞咽：这一阶段的宝宝可以开始接受肉类食物，但这并不表明宝宝的消化功能已经接近成人了。仍然要坚持辅食添加循序渐进的总体原则。除继续熟悉各种食物的新味道和感觉外，还应该逐渐改变食物的质感和颗粒大小，逐渐从泥糊状食物向固体食物过渡，使辅食取代一顿奶而成为独立的一餐，同时锻炼宝宝的咀嚼能力。

【10-12个月】建议规律饮食模式：虽然宝宝的牙还没有几颗，但已经会用牙床咀嚼食物，这一时期，让宝宝充分练习咀嚼尤其重要。此时，宝宝已经进入换奶晚期，每日应继续母乳喂养并吃3次主食和1次点心。可以适当增加食物的硬度。

【添加辅食后腹泻】

在给宝宝添加辅食的初期，会出现对某种食物不耐受的现象，起着腹泻是最常见的表现之一。遇到这种情况如何不是很严重，可以继续食用，仔细观察3天；如果情况逐渐好转，则可以坚持到恢复正常后，再添加新的食物。但是如果继续加重，则应暂停此种辅食，等宝宝恢复后过段时间在进行尝试，还是出现类似情况，应更换其他辅食。

【添加辅食后便秘】

如果添加辅食后，宝宝出现大便干结、排便费力等情况，则有可能是便秘引起的。宝宝便秘是较为常见的问题，可以从增加肠道菌群，摄入足够的纤维素两个方面进行预防和治疗。粗粮是膳食纤维的最佳来源，宝宝的辅食加工过细、过精，虽然有利于营养的吸收，但是食物残渣少，容易使纤维素摄入不足，引起便秘。

【添加辅食后过敏】

添加辅食的过程中，家长们要注意观察孩子对待食物的态度，以便及时的发现过敏源。如果宝宝一见某种食物就特别排斥，或进食初期正常，但很快出现抗拒等情况，除了是宝宝对食物的味道和形态不接受，最先考虑的应该是食物过敏或不耐受的问题。如果宝宝食用某种食物后出现恶心或呕吐，则有可能是过敏引起的。怀疑食物过敏可以先停止食用可疑过敏食物，添加辅食的过程不要太快，可以一种一种地添加，每次添加后先观察三天，看宝宝有无不适反应。过敏反应会在1-3天内反应出来，如果确定确定宝宝队某种食物过敏，应停止食用此种过敏食物至少3个月；也不要过于担心会因缺少某种食物而引起营养缺乏，只要可以合理搭配，营养就不会缺失。
